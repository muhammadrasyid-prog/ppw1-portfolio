Pengenalan HTML

HTML merupakan bahasa markup yang digunakan untuk membuat struktur halaman web. Dengan HTML kita dapat menambahkan teks, gambar, link, dan berbagai elemen lainnya.

Pengenalan CSS

CSS digunakan untuk mengatur tampilan website seperti warna, ukuran teks, layout, dan jarak antar elemen agar halaman web terlihat lebih menarik.
